```
Color palette (strict):

- soft gold (#b8935a) — accent, brass tones
- twilight purple (#2a1f3a) — ambient depth at edges
- deep navy ink (#0a1f2e) — shadow + typography
- accent warm (#d4b87a) — highlight on metallic

NEGATIVE PROMPT (never include):
- human faces, patients, recognizable people
- medical clutter, exposed wires, plastic equipment
- neon signs, aggressive saturation, blown-out highlights
- logos, brand names, text overlays
- cliché stock-photography poses

Output: 4K, 16:9 unless specified, RAW photograph quality.
```

## ② LOGO PROMPT (for Nano Banana Pro)

> User note: you said you want to KEEP Bright's original facade logo (8-petal geometric mark). This prompt generates a **digital-asset version** of the same motif, in pure SVG-friendly form, for use as a favicon / nav logo / journal masthead — without recreating their actual entrance signage.

```
STYLE LOCK [above]

Generate a single digital mark for Bright Specialized Clinics:

An 8-petal geometric Najdi-style star/floral motif, inscribed within
an invisible circle. Each petal is symmetric and meets at a central
point. The petals are subtly diamond-shaped, not rounded.

Material rendering: brushed warm gold (#b8935a) with subtle
metallic gradient — darker gold at outer edges, lighter
champagne gold (#d4b87a) at the center hub.

Background: pure cream (#f8f6f0), absolutely flat, no shadow except
a single 2px-equivalent soft drop shadow directly below the mark
in 8% navy.

The mark itself should read clearly at:
- 32px favicon
- 88px nav logo
- 240px journal masthead

NO outer border ring. NO inner text. NO stylization beyond the
geometric petal pattern.

Format: square 1024×1024 PNG with full transparency, plus a second
version rendered as solid gold on cream background.
```

**Iteration variants** to try if first output is off:
1. Replace "Najdi-style" with "Islamic geometric tessellation"
2. Replace "8-petal" with "8-point" (if petals come out too floral)
3. Try "minimal geometric star with 8 equal points, brushed gold"
4. For the cream version: add "logo lockup, isolated, no decorative elements"

## ⑦ Generation workflow (~25 min for full Bright set)

```
1. Open gemini.google.com (logged in with paid Gemini Pro)
2. Paste the STYLE LOCK once at top of conversation
3. Then for each prompt above:
   a. Paste the specific prompt
   b. Wait ~12s for first render
   c. If off — say "render variant 2 with [adjustment]"
   d. Pick best of 2-3 outputs
   e. Click download → save as PNG/JPG

4. For each generated image:
   a. Drag to https://squoosh.app
   b. Convert to AVIF, quality 85%
   c. Save with naming: bright-[department].avif

5. Drop AVIFs into bright-al-ahsa-clinic/public/hero/

6. Update component references — I'll give you those file-by-file
   the moment you have images ready.
```
